Criminal Defense System Collapse

The Lavali Protocol & Public Defender Crisis

The Massachusetts criminal justice system is facing a functional collapse under the "Lavali Protocol." The state is currently operating without a functioning criminal defense system due to a mass work stoppage by bar advocates.

The Lavali Protocol

This protocol requires that if the state cannot provide a lawyer to an indigent defendant within specified timeframes, the case must be dismissed:

7 Days For defendants in jail Case dismissed if no lawyer assigned
45 Days For defendants on bail Case dismissed if no lawyer assigned

Result: Thousands of criminal cases are being dismissed‚Äînot because defendants are innocent, but because the state cannot fulfill its constitutional obligation to provide legal counsel.
